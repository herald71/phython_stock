## 1. 기술적 지표 (Technical Indicators)

프로그램은 CSV/Excel 데이터를 로드한 후 자동으로 다음 기술 지표들을 계산합니다.

| 지표 | 계산 방법 | 용도 |
|------|----------|------|
| **MA5 / MA10** | 5일·10일 단순이동평균 | 단기 추세 파악 |
| **MA20** | 20일 단순이동평균 | 중기 추세 & 지지/저항 기준선 |
| **MA60** | 60일 단순이동평균 | 장기 추세 판단 |
| **RSI (14일)** | `100 - 100/(1+RS)` (RS = 14일 평균상승폭 / 평균하락폭) | 과매수(≥70) / 과매도(≤30) 판단 |
| **MACD** | EMA12 - EMA26 | 추세 전환 감지 |
| **Signal** | MACD의 9일 EMA | MACD 크로스 기준선 |
| **MACD Histogram** | MACD - Signal | 추세 강도 시각화 |
| **골든크로스** | MA20이 MA60을 상향 돌파 | 매수 신호 |
| **데드크로스** | MA20이 MA60을 하향 돌파 | 매도 신호 |
| **볼린저 밴드** | MA20 ± 2σ (20일 표준편차) | 변동성 기반 매매 |
| **터틀 채널** | 20일 최고가 / 10일 최저가 | 돌파 매매 기준 |
| **거래량 MA20** | 20일 평균 거래량 | 거래량 급증 감지 |
| **모멘텀(단기)** | 20일 수익률 | 단기 추세 강도 |
| **모멘텀(장기)** | 60일 수익률 | 장기 추세 강도 |

## 2. 종합 매매 추천 시스템

5가지 관점에서 각각 **-2 ~ +2점**을 부여하고, 합산하여 최종 추천을 결정합니다.

### 2.1 점수 산정 기준

#### ① RSI 기반
| 조건 | 점수 | 해석 |
|------|------|------|
| RSI ≤ 20 | **+2** | 극도의 과매도 → 강한 매수 |
| RSI ≤ 30 | **+1** | 과매도 → 반등 가능 |
| 30 < RSI < 70 | **0** | 중립 |
| RSI ≥ 70 | **-1** | 과매수 → 조정 가능 |
| RSI ≥ 80 | **-2** | 극도의 과매수 → 강한 매도 |

#### ② MACD 기반
| 조건 | 점수 | 해석 |
|------|------|------|
| MACD 골든크로스 발생 (히스토그램 음→양) | **+2** | 강한 매수 |
| MACD > Signal | **+1** | 상승 추세 |
| MACD 데드크로스 발생 (히스토그램 양→음) | **-2** | 강한 매도 |
| MACD < Signal | **-1** | 하락 추세 |

#### ③ 이동평균 크로스
| 조건 | 점수 | 해석 |
|------|------|------|
| 당일 골든크로스 발생 | **+2** | 중장기 상승 전환 |
| MA20 > MA60 | **+1** | 상승 추세 유지 |
| 당일 데드크로스 발생 | **-2** | 중장기 하락 전환 |
| MA20 < MA60 | **-1** | 하락 추세 유지 |

#### ④ 가격 위치 (현재가 vs MA20)
| 조건 | 점수 | 해석 |
|------|------|------|
| MA20 대비 -5% 이하 | **+1** | 반등 가능 |
| MA20 근처 (±5%) | **0** | 중립 |
| MA20 대비 +5% 이상 | **-1** | 단기 과열 |

#### ⑤ 모멘텀 (최근 5일)
| 조건 | 점수 | 해석 |
|------|------|------|
| 5일간 +3% 이상 상승 | **+1** | 단기 모멘텀 양호 |
| 5일간 ±3% 이내 | **0** | 방향성 불분명 |
| 5일간 -3% 이상 하락 | **-1** | 단기 모멘텀 약세 |

### 2.2 최종 추천 결정

| 종합 점수 | 추천 | 설명 |
|-----------|------|------|
| **+3 이상** | 🔴 **강력 매수** | 여러 지표가 매수를 가리킴 |
| **+1 ~ +2** | 🟠 **매수 대기** | 매수 우세지만 확인 필요 |
| **0** | ⚪ **중립 (관망)** | 뚜렷한 방향 없음 |
| **-1 ~ -2** | 🟣 **매도 대기** | 매도 우세지만 확인 필요 |
| **-3 이하** | 🔵 **강력 매도** | 여러 지표가 매도를 가리킴 |

## 3. 백테스팅 전략 (9가지)

모든 전략은 **초기 투자금 전액 매수/전량 매도** 방식으로 시뮬레이션됩니다.  
결과를 **매수-보유(Buy & Hold) 전략**과 비교하여 초과수익률을 측정합니다.

---

### 전략 1: 골든/데드크로스 전략

> **유형**: 추세 추종  
> **핵심 지표**: MA20, MA60

| 신호 | 조건 |
|------|------|
| **매수** | `Golden == True` (MA20이 MA60 상향 돌파) |
| **매도** | `Death == True` (MA20이 MA60 하향 돌파) |

- **장점**: 중장기 추세 전환을 안정적으로 포착
- **단점**: 횡보장에서 잦은 뒤늦은 신호 (후행성)

---

### 전략 2: RSI 전략

> **유형**: 역추세(평균 회귀)  
> **핵심 지표**: RSI (14일)

| 신호 | 조건 |
|------|------|
| **매수** | RSI ≤ 30 (과매도) |
| **매도** | RSI ≥ 70 (과매수) |

- **장점**: 극단적 과매수/과매도 구간에서 반전 포착
- **단점**: 강한 추세장에서 조기 매도 / 조기 매수 위험

---

### 전략 3: MACD 전략

> **유형**: 추세 추종 + 모멘텀  
> **핵심 지표**: MACD Histogram

| 신호 | 조건 |
|------|------|
| **매수** | MACD 히스토그램이 음수 → 양수로 전환 |
| **매도** | MACD 히스토그램이 양수 → 음수로 전환 |

- **장점**: 추세 전환 시점을 비교적 빠르게 포착
- **단점**: 히스토그램 노이즈로 잦은 거래 발생 가능

---

### 전략 4: 종합 전략

> **유형**: 멀티 팩터  
> **핵심 지표**: RSI + 골든/데드크로스 + MA 정배열/역배열

**매수 조건** (buy_score ≥ 2):

| 조건 | 점수 |
|------|------|
| RSI ≤ 30 | +1 |
| 골든크로스 발생 | +2 |
| MA20 > MA60 | +1 |

**매도 조건** (sell_score ≥ 2):

| 조건 | 점수 |
|------|------|
| RSI ≥ 70 | +1 |
| 데드크로스 발생 | +2 |
| MA20 < MA60 | +1 |

- **장점**: 여러 지표를 종합하여 **거짓 신호 필터링**
- **단점**: 조건이 까다로워 거래 빈도가 낮을 수 있음

---

### 전략 5: 볼린저 밴드 전략

> **유형**: 평균 회귀 (변동성 기반)  
> **핵심 지표**: BB_Upper, BB_Lower (MA20 ± 2σ)

| 신호 | 조건 |
|------|------|
| **매수** | 종가 ≤ 하한 밴드 (BB_Lower) |
| **매도** | 종가 ≥ 상한 밴드 (BB_Upper) |

- **장점**: 변동성이 큰 횡보장에서 효과적
- **단점**: 밴드 밖으로 추세가 지속되면 손실 확대

---

### 전략 6: MA 돌파 전략

> **유형**: 추세 추종  
> **핵심 지표**: 종가 vs MA20

| 신호 | 조건 |
|------|------|
| **매수** | 전일 종가 ≤ MA20 → 당일 종가 > MA20 (상향 돌파) |
| **매도** | 전일 종가 ≥ MA20 → 당일 종가 < MA20 (하향 이탈) |

- **장점**: 단순하고 직관적, 빠른 반응 속도
- **단점**: 횡보장에서 잦은 휩소(Whipsaw) 발생

---

### 전략 7: 거래량 급증 전략

> **유형**: 거래량 분석  
> **핵심 지표**: Volume / Vol_MA20, 양봉/음봉 판별

| 신호 | 조건 |
|------|------|
| **매수** | 거래량 ≥ 20일 평균의 **2배** + **양봉** (Close > Open) |
| **매도** | 거래량 ≥ 20일 평균의 **2배** + **음봉** (Close < Open) |

- **장점**: 세력/기관의 대량 매매를 포착
- **단점**: 거래량 급증이 드물어 거래 빈도가 매우 낮음

---

### 전략 8: 터틀 트레이딩 전략

> **유형**: 돌파 매매 (Breakout)  
> **핵심 지표**: 20일 최고가, 10일 최저가  
> **원조**: 리처드 데니스의 '터틀 트레이딩' 실험에서 유래

| 신호 | 조건 |
|------|------|
| **매수** | 종가 > 전일 20일 최고가 (신고가 돌파) |
| **매도** | 종가 < 당일 10일 최저가 (신저가 이탈) |

- **장점**: 강한 추세를 타고 큰 수익 가능
- **단점**: 거짓 돌파(False Breakout) 시 손실, 비대칭 진입/이탈(20일 vs 10일)

---

### 전략 9: 듀얼 모멘텀 전략

> **유형**: 모멘텀  
> **핵심 지표**: 20일 수익률(단기), 60일 수익률(장기)  
> **원조**: 게리 안토나치(Gary Antonacci)의 '듀얼 모멘텀' 이론 응용

| 신호 | 조건 |
|------|------|
| **매수** | 단기 모멘텀 > 0 **AND** 장기 모멘텀 > 0 |
| **매도** | 단기 모멘텀 < 0 |

- **장점**: 상승 모멘텀이 확인된 종목에만 투자하는 **보수적** 전략
- **단점**: 모멘텀 전환 시점에 후행, V자 반등에 늦게 진입

## 5. 백테스팅 시뮬레이션 방식

```
초기 상태: cash = 초기투자금, shares = 0

매일 반복:
  1. 선택한 전략의 규칙으로 BUY / SELL 신호 판단
  2. BUY 신호 & 현금 보유 중 → 전액 매수 (shares = cash / price)
  3. SELL 신호 & 주식 보유 중 → 전량 매도 (cash = shares × price)
  4. 포트폴리오 가치 = cash + (shares × 현재가)
  5. 매수-보유 기준값 = (초기투자금 / 첫날 종가) × 현재가
```

**결과 비교 지표**:
- 전략 수익률 vs 매수-보유 수익률
- 초과수익률 (전략 - 매수-보유)
- 총 거래 횟수
